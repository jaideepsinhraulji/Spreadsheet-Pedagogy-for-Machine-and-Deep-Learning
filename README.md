# Machine Learning Mechanics in Excel

This repository hosts interactive, spreadsheet-based implementations of classic Machine Learning algorithms. By unrolling the mathematical steps directly inside Excel, these sheets serve as pedagogical tools to help students visualize foundational ML theory without hiding behind library abstractions.

## 📊 Contained Templates

### 1. Naive Bayes Classifier (`Naive_Bayes_Classifier_Car_Stolen.xlsx`)
* **Objective:** Predict whether a vehicle will be stolen based on categorical features: `Color`, `Type`, and `Origin`.
* **Pedagogical Core:** Demonstrates explicit Frequency Mapping, Likelihood Calculations, and the application of the **Bayes Chain Rule** for final posterior determination.

## 🚀 How to Use These Files
Because these are interactive spreadsheets containing formulas, they cannot be fully previewed directly inside the GitHub browser interface. 

**To interact with the models:**
1. Click on the `.xlsx` file name above.
2. Click the **Download** button on the right side of the file pane to save it to your local machine.
3. Open the file in **Microsoft Excel** or **Google Sheets** to explore the cell formulas and test new prediction traces (e.g., modifying row 11).

### 2. Decision Tree Classifier & Regressor (`Decision Tree.xlsx`)
* **Objective:** Map structural decision paths for classification and regression using step-by-step mathematical criteria.
* **Pedagogical Core & Worksheets:**
  * `Ori-E`: Demonstrates **Entropy** calculations and **Information Gain** optimization for splitting attributes.
  * `Ori-G`: Unrolls **Gini Impurity** parameters to contrast with the entropy methodology.
  * `Cont D`: Illustrates how **continuous features** are handled via dynamically sorted thresholds and binary candidate splits.
  * `Regression`: Steps out of classification to show Variance Reduction and Mean Squared Error (MSE) minimization for quantitative targets.
  * `Final TREE`: Visualizes the compiled logical execution tree resulting from the underlying manual arithmetic.
